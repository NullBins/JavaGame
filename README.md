# 미지의 던전 탐험기

Java 콘솔 환경에서 실행되는 간단한 던전 탐험 게임입니다.

## 실행 방법

터미널에서 다음 명령어를 입력합니다.

```bash
javac AdventureGame.java
java AdventureGame
```

## 게임 시작

게임을 실행하면 시작 방식을 선택할 수 있습니다.

```text
1. 처음부터 시작한다.
2. 이어서 시작한다.
```

* **처음부터 시작**: 플레이어 이름을 입력하고 새로운 게임을 시작합니다.
* **이어서 시작**: 저장된 플레이어 데이터를 불러와 게임을 계속합니다.

## 게임 메뉴

마을에서 다음 행동을 선택할 수 있습니다.

```text
1. 던전 탐험
2. 포션 구매
3. 여관에서 휴식
4. 저장하고 게임 종료
```

### 던전 탐험

던전에 들어가면 다음 이벤트 중 하나가 발생합니다.

* 몬스터 전투
* 보물상자 발견
* 함정 발생

### 포션 구매

* 가격: 30골드
* 포션 사용 시 HP 40 회복
* 최대 HP는 100입니다.

### 여관

* 가격: 20골드
* HP 50 회복
* 최대 HP는 100입니다.

## 저장 기능

마을에서 `4번`을 선택하면 현재 플레이어 정보가 저장되고 게임이 종료됩니다.

저장되는 정보는 다음과 같습니다.

* 플레이어 이름
* HP
* 골드
* 포션 개수

저장 데이터는 `player_data.txt` 파일에 기록됩니다.

게임을 다시 실행한 후 `이어서 시작`을 선택하면 저장된 상태에서 계속 플레이할 수 있습니다.

## 주의사항

게임을 정상적으로 저장하려면 마을 메뉴에서 반드시 `4번 저장하고 게임 종료`를 선택해야 합니다.
